1. Navbar

This feature of prebuilt navbars in Bootstrap is a very handy feature for every website that needs a navigation bar.

Here is where we got this code, but we also tweaked it and removed some things.

The overall navbar is spanned by a navbar class. The background color (navy) is set by adding it in the style attribute in the nav tag. The color scheme of dark is set by adding "navbar-dark" in the class attribute of the nav tag. Using "container-fluid" allows the navbar to span the entire width. Using "navbar-nav" styles the navbar so that all the items are in a row. This is used in the class attribute of the ul tag. This is also used to align all the items to the left and resizes the navbar properly. "nav-item" is used for adding items to the navbar. "nav-link" is used to add links to the navbar in the same style as the rest of the navbar. "active" and "disabled" can be used to specify the properties of these links.

If you make the window smaller (and it is encouraged to do so!), the links collapse on top of each other.

2. Grids

This feature is best used for organizing elements of a page in a specific format. In this case, the chunks of text are aligned next to each other

Here is where we got the code. We changed the text.

This is a very simple example, but look at how the columns are all next to each other. This is done through the adding "container" to the class attribute in the div tag.

If you resize the window (please do!) you can see that the columns stay next to each other and don't collapse on top of one another.

This is a column.
Adding "row" to the class attribute of the div tag allows the columns to all appear next to each other.
The style of the columns are governed by adding "col" to the class attribute of the div tag.

3. Cards

This next feature is best used for displaying information concisely as well as putting a link to redirect the user elsewhere if needed.

Here is where we got the code. We removed the buttons and changed the text.

Again, rows and columns are used to organize these cards. However, we will focus on the cards themselves.

Cards basically appear how you would expect them to. They have light borders and contain elements within them. Adding "card" in the class attribute of the div tag creates a card. "card-body" organizes the elements in the card. "card-title" styles the card's title. "card-text" styles the card's text below the title.
